# 🌈 Système de Couleurs OKLCH

**Espace colorimétrique perceptuel uniforme, optimal pour l'accessibilité médicale**

<div style={{ 
  padding: '1.5rem', 
  backgroundColor: 'hsl(var(--primary) / 0.05)', 
  border: '1px solid hsl(var(--primary) / 0.2)',

}}>
  <h3>🔬 Pourquoi OKLCH ?</h3>
  <p>
    Contrairement au HSL traditionnel, <strong>OKLCH</strong> garantit une luminosité perceptuelle constante 
    et des contrastes prévisibles, essentiels pour l'accessibilité des personnes âgées et handicapées.
  </p>
  <ul>
    <li><strong>L</strong> = Lightness (0-100%) - Luminosité perceptuelle uniforme</li>
    <li><strong>C</strong> = Chroma - Saturation colorimétrique</li> 
    <li><strong>H</strong> = Hue (0-360°) - Teinte sur la roue des couleurs</li>
  </ul>
</div>

### Validation de Contraste

```javascript
// Outil de validation (exemple)
const validateContrast = (foreground, background) => {
  const ratio = calculateContrast(foreground, background);
  return ratio >= 4.5 ? 'WCAG AA ✓' : 'WCAG AA ❌';
};

// Exemples validés
validateContrast('var(--color-primary-700)', 'white');  // ✓ 5.83:1
validateContrast('var(--color-error-700)', 'white');    // ✓ 6.89:1
validateContrast('var(--color-neutral-900)', 'white');  // ✓ 15.36:1
```
